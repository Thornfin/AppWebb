## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npx expo start
   ```

3. Open the app in your preferred environment:
   - iOS Simulator
   - Android Emulator
   - Web Browser
   - Expo Go app on your physical device
